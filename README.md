# Coin Flipper (Python)

This project simulates flipping a coin using Python.  
It started as a basic random flip and gradually evolved into a structured mini project.

---

## Features

- Single coin flip
- Multiple coin flips
- Heads and Tails counter
- Probability calculation
- User input support
- Input validation

---

## How It Works

1. User enters number of flips.
2. Program simulates random coin flips.
3. Counts total Heads and Tails.
4. Calculates probability percentage.
5. Displays clean formatted results.
